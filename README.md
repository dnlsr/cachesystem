## Система кэширования на PHP
Класс для кэширования данных с поддержкой двух драйверов хранения: файловый и в памяти.
# Основные возможности:
	Сохранение, получение, удаление данных;
	Поддержка времени жизни (TTL);
	Два драйвера: FileCache и MemoryCache.
# Запуск тестов
	composer install
	php tests/CacheTest.php
	php index.php
